Case Study

A User Experience for UCF: Student Access Resource Center

A conceptualization into material view

By FRANCIS CACCAVALE

A holistic-approach design as a user experience to help a university increase undergraduate pass rates, especially for introductory courses given to first year students.

Empathize

Given how the heights of both society and Internet have come to integrate, an obvious scenario was to question how the individual student and Internet integral may configure in an existing university culture, respectfully.

Twenty sophomore students were given a questionaire via Google Forms with some example questions being:

  • Which social-network resonates with you most?
  • Did social-networking online create academic distractions your first semester? If yes, why?
  • Has social-networking online help create interactions with direct peers or classmates thus far? If yes, how?
  • What negatives do you identify as to broad social-networks online?

After synthesizing results, common insights were:

  • Transitioning into a cultured university creates a sense of shellshock or forms an edge
  • Social-media online is a broad paint stroke

Define

Ultimately, pain points were surrounding an apex at Internet and university culture. A social-networking pivot was necessary to filter the needs of a particular user set, being incoming freshmen, and a recalibration of the Internet and university culture integral.

Ideate

An example of an ideation technique utilised in this design was the ‘how might we…’ exercise. A ‘how might we…’ exercise begins with a problem statement only to break that larger challenge into smaller actionable pieces, as a divide and conquer approach.

  • How might we create an interface to help faculty identify/empathize with students?
  • How might we let this interface be a user experience design style 'stickersheet'?
  • How might we let each student personalize the UX design style 'stickersheet' to their liking?
  • How might we create an internal social-network with these 'stickersheet' uploads?

As a resolve via sole concept and design, each student will create a UX design style ‘sticker-sheet’ which would instead feature personal attributes; whether that be a color palette, interests in their favorite typeface, a graph of their recent Spotify playlist and/or photographs instead of logos. This aggregate of stickersheets would then compile into an internal social network to create an economy of empathy and ultimately help passing rates.
